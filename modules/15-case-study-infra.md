# Interactive case study: Securing endpoints and infrastructure

- Module
- 5 Units

 Advanced | Security Engineer | Solution Architect | Microsoft Intune | Microsoft Defender XDR | Microsoft Azure Defender for IoT | Microsoft Defender for Cloud | Microsoft Defender for Endpoint | Azure Arc

Apply your cybersecurity architect skills on a real business scenario focused on endpoint and infrastructure security. Analyze design requirements, answer conceptual and technical questions and design a solution to meet the business needs.

## Learning objectives

In this interactive case study, you:

- Explore common security gaps in distributed, hybrid enterprises spanning endpoints, OT/IoT, and cloud governance.
- Learn how to analyze threats and map Microsoft security capabilities to address risks across IT and industrial environments.
- Apply architectural reasoning to design and sequence a Zero Trust-aligned solution that secures endpoints and infrastructure end-to-end.

## Prerequisites

- Experience and knowledge in identity and access, platform protection, security operations, securing data and securing applications.
- Experience with hybrid and cloud implementations.
